# SukashiMaker

https://choyanglue.github.io/SukashiMaker/sukashimaker.html

SukashiMaker は、写真に含まれる EXIF 情報を読み取り、カメラ名や撮影設定を画像下部へ整ったレイアウトで描画するシンプルな Web ツールです。

## 主な機能

- JPEG などの EXIF 付き画像を読み込んでプレビューを生成
- カメラメーカーや機種名、焦点距離、F値、シャッタースピード、ISO を表示
- 生成した画像を PNG として保存
- スマホと PC の両方で見やすいレスポンシブレイアウト
- 日本語 / English の画面表示切り替え

## 使い方

1. [`sukashimaker.html`](https://choyanglue.github.io/SukashiMaker/sukashimaker.html) をブラウザで開きます
2. 画像ファイルをアップロードします
3. プレビューを確認します
4. 必要に応じて `画像を保存 / Save image` で保存します

## 補足

- EXIF 情報が含まれていない画像では、撮影情報を表示できません
- 初期表示の言語はブラウザの設定に合わせて自動選択されます
- 画面右上のボタンから日本語と英語を切り替えられます
